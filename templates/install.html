Install

Nix

Multi-user:

$ sh <(curl -L https://nixos.org/nix/install) --daemon

Single-user:

$ sh <(curl -L https://nixos.org/nix/install) --no-daemon

Which installer should you chose? We reccomend multi-user, however it requires root access.
Run single-user if root access is not available.
